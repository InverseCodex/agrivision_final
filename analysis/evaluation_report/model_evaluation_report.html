How The Model Is Computed (Transparent Processing)

  1. Image RGB channels are summarized into mean_red, mean_green, mean_blue, and green coverage.
  2. Vegetation proxies are computed: VARI, GLI, NGRDI, ExG.
  3. Weighted score model combines normalized proxies: VARI(0.30), GLI(0.25), NGRDI(0.20), ExG(0.25).
  4. Context penalties adjust score (low rainfall / extreme temperature).
  5. Score is mapped to health bands: healthy/watch/stressed/critical.
  6. Confidence is derived from proxy spread and bounded by model limits.

Extended indices used for operations planning in this project: TGI, MGRVI, and LAI proxy plus canopy/stress/biomass/yield proxies.

Risks, Limits, and Mitigations

  • RGB-only limitation: This is a proxy to multispectral indices; mitigate by field scouting in critical zones.
  • Lighting sensitivity: Enforce consistent capture time and weather windows.
  • Domain shift: Different crop stages and camera settings can move score distributions.
  • Recommendation safety: Treat recommendations as decision support, not autonomous farm control.
